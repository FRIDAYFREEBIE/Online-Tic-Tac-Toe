# Online Tic-tac-toe

### 🌐 Senior Year First Semester Game Network Project at Korea Animation High School

---

## 📌 Project Overview
- **Project Concept**: Online Tic-tac-toe
- **Development Period**: Approximately 2 Days
- **Goal**: To gain experience in network programming

---

## 🔑 Key Technologies

- **Unity**
  - Game engine used for gameplay implementation and multiplayer systems

- **C#**
  - Core programming language used for gameplay and networking logic

- **Photon**
  - Implemented real-time turn-based multiplayer gameplay
  - Managed room creation, player synchronization, and turn-based game flow

---

## 🤔 What I Learned

- This was my second online multiplayer game project using the Photon server system.
  By applying experience gained from my first Photon project, I was able to complete the project successfully within a short development period of only two days.

- Previous multiplayer development experience significantly improved both development speed and workflow efficiency.

- Although the game itself was simple, implementing room creation, turn-based gameplay logic, and real-time synchronization helped reinforce the importance of client-server architecture and network data synchronization.
